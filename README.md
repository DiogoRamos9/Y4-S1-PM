## 📋 About This Project

This repository contains the **Project Management Report** for *Carrier Invest, SA*'s I&D (Research & Development) initiative — a digital transformation project focused on modernizing legacy IT systems through Big Data, Cloud infrastructure, and Agile/Waterfall hybrid project management practices.

Developed as part of a university project at **U.Porto**, this report documents the full lifecycle of planning and managing a large-scale (36-month) technology modernization project for a real-world logistics business unit.

### 🔍 What's Inside

- **Strategic Objectives** — Technological modernization, data quality & structuring, data-driven decision support, and AI-readiness for a logistics-focused company.
- **Project Methodology** — A hybrid Waterfall + Agile approach, justified and mapped across 7 core activities (A1–A7).
- **Vision Scope** — In-scope and out-of-scope definitions, focused on the WMS (Warehouse Management System) and Order-to-Cash processes.
- **Gantt Chart & Milestones** — 36-month roadmap with dependency logic (FS, SS, FF relationships) and 4 key milestones.
- **Stakeholder Analysis** — Influence/Importance matrix and engagement strategies for all key stakeholders.
- **Organizational Structure** — Team layers (Strategic, Tactical, Operational, Support) and required technical profiles (Data Scientists, Cloud Architects, Integration Engineers, etc.).
- **Risk Management** — Full risk register (R1–R12) with probability/impact scoring and mitigation plans.
- **Communication Plan & Governance Model** — RACI matrix and recurring governance forums (Steering Committees, Decision Team Meetings, etc.).
- **Jira Project Setup** — Epics, tasks, user stories, releases, sprint examples, and Kanban board configuration for monitoring execution.
- **KPIs & Metrics** — Lead vs. Lag indicators across technology, project execution, adoption, and business outcomes, plus SMART goal examples.
- **Benefit Tracker** — Strategic benefit tracking table tied to baseline vs. target values.
- **Earned Value Analysis (EVA)** — Framework for tracking schedule (SPI) and cost (CPI) performance.



*Project developed for the Project Management course at Universidade do Porto.*
